# Analysis of Quicksort and Randomized Quicksort
The project’s aim is to determine which algorithm, Quicksort or Random Quicksort, is better under certain circumstances. It will also explore the results while using Hoare’s Partitioning and Lomuto’s Partitioning. This is important in computer networks as it helps programmers determine what algorithm is best for their particular needs, rather than just choosing something that works but is less efficient in the long run. This report details the information and results gathered thus far. It also describes the future plans for finalizing the data and exploring more variables. 

## To run the code:
- Run "make" in terminal
- Run "make clean" to remove unwanted object files
- run " ./main " in terminal
- input desired array size
- Review results
